# WhisperJAV-Auto 使用说明

WhisperJAV-Auto 是一个 Windows PowerShell 自动化封装，用于：

```text
视频 -> WhisperJAV Ensemble 双 Pass 日语转录 -> DeepSeek 中文翻译 -> 视频同名 .srt
```

## 目录结构

用户日常只需要关注根目录这些文件：

- `开启中文字幕自动处理.bat`：开启自动处理并打开任务进度窗口。
- `关闭中文字幕自动处理.bat`：关闭自动处理并停止相关进程。
- `查看中文字幕处理状态.bat`：查看当前任务、最近日志和失败清单数量。
- `config.example.json`：公开配置模板。
- `config.json`：本机私有配置文件，不建议提交到 Git。
- `使用说明.md`：本文档。

`internal` 是内部脚本目录，正常不要修改。

## 首次配置

1. 复制配置模板：

```powershell
Copy-Item .\config.example.json .\config.json
```

2. 编辑 `config.json`。

主要修改 `paths.videoRoots`、`paths.workRoot`、`paths.whisperJavInstallDir`。

```json
"paths": {
  "workRoot": "E:\\WhisperJAV-Auto\\work",
  "manualOutputDir": "D:\\WhisperJAV-Auto\\manual-output",
  "videoRoots": [
    "X:\\videos"
  ],
  "whisperJavInstallDir": "D:\\whisperjav"
}
```

建议：

- `workRoot` 放本机磁盘，不要放 NAS。
- `videoRoots` 填实际视频根目录。
- `manualOutputDir` 用于目标目录不可写时保留字幕。

## DeepSeek 配置

推荐使用环境变量保存 API Key：

```powershell
setx DEEPSEEK_API_KEY "sk-xxxxxxxx"
```

也可以直接写入 `config.json`：

```json
"deepSeek": {
  "apiKey": "sk-xxxxxxxx"
}
```

公开仓库不要提交带真实 API Key 的 `config.json`。

## 日常使用

开启：

```text
双击 开启中文字幕自动处理.bat
```

关闭：

```text
双击 关闭中文字幕自动处理.bat
```

查看状态：

```text
双击 查看中文字幕处理状态.bat
```

## 跳过规则

脚本会跳过：

- 已有同名外挂字幕的视频。
- MKV 内嵌字幕的视频。
- 文件名包含中文字幕标记的视频，例如 `ABF-159-C.mp4`、`CAWD-590-C_GG5.mp4`、`NSFS-400ch.mp4`。
- 失败清单里的损坏视频。

## 日志与输出

日志目录：

```text
logs
process-logs
```

正常输出：

```text
影片.mp4
影片.srt
```

如果目标视频目录不可写，中文字幕会保留到：

```text
manual-output
```
